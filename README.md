# 워커힐김치 ERP

워커힐 김치 주문·고객·재고 관리 시스템 (PWA)

## 🚀 배포 방법

### 1단계: GitHub 업로드
1. github.com 접속 → 로그인
2. 우측 상단 `+` → `New repository`
3. Repository name: `walkerhill-kimchi-erp`
4. `Public` 선택 → `Create repository`
5. "uploading an existing file" 클릭
6. 이 폴더의 모든 파일을 드래그 앤 드롭
7. `Commit changes` 클릭

### 2단계: Vercel 배포
1. vercel.com 접속 → Continue with GitHub
2. `Add New...` → `Project` 클릭
3. `walkerhill-kimchi-erp` 선택 → `Import`
4. 기본 설정 그대로 → `Deploy` 클릭
5. 3분 대기 → 완료!

### 3단계: 크롬에서 설치
1. 생성된 URL을 크롬에서 열기 (예: `walkerhill-kimchi-erp.vercel.app`)
2. 주소창 오른쪽 설치 아이콘 클릭
3. `설치` 클릭 → 바탕화면에 아이콘 생성됨!

## 💻 로컬 실행 (개발자용)

```bash
npm install
npm run dev
```

## 📦 빌드

```bash
npm run build
```

빌드 결과는 `dist/` 폴더에 생성됩니다.
